A Developing Developer

Currently re-reading:
Bitwise: A Life in Code

I read this book before I started my coding bootcamp at Coding Dojo. I'd been studying algorithms on AppAcademy's online Bootcamp Prep program. I could code the algorithm to FizzBuzz, but I had no idea how complex algorithms could get.

The computer science component of software engineering was completely foreign to me. I'd known basic HTML and CSS, but concepts of data structures such as arrays, binary search trees, singly-linked lists, stacks, and queues were completey outside of my realm of knowledge.

Re-reading this book after going through a coding bootcamp has given me a completely different perspective. In chapter one, the author, David Auerbach, talks about assembly language:

"When I took a compiler class in college, the infrastructure of the computer opened up to me. There was no longer a miracle in between my code and its execution. I could see the whole picture, finally, and it was beautiful."

After finshing the coding bootcamp, I feel like I've uncovered a small portion of the whole picture. I know I have much to learn, but attending a coding bootcamp has given me the ability to see the inner-workings of the web and how all of it's component parts fit together.
